# WTWR — Frontend (React + Vite)

A clean React frontend for **What To Wear (WTWR)**. It shows weather-based clothing suggestions and supports **Sign Up / Log in**, a protected **Profile** page, likes, and profile editing — all styled with **BEM**.

## Quick Start

```bash
# 1) Install dependencies
npm install

# 2) Start dev server
npm run dev

# 3) Build for production
npm run build

# 4) Preview the production build
npm run preview
```
